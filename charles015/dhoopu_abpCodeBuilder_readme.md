# abpFrame_Angular

#### 介绍
基于abp angular版本的开发框架，参考了asp.net zero,magicodes.admin和52abp,学习用,配套也开发了个代码生成器，代码生成器介绍博客：https://www.cnblogs.com/L_tommy/p/10872389.html，
不过当前都还是初始版本，后面有时间继续完善

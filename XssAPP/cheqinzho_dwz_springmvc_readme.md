开发环境:

	JDK1.6或以上
	Tomcat6或以上
	Eclipse IDE for Java EE
	Mysql 5.0以上版本

Quick Start:

	1) 下载源码 https://code.csdn.net/dwzteam/dwz_springmvc
	2) 数据库：Mysql
		请修改 src/jdbc.properties这个文件的数据库连接
		创建数据库dwz_springmvc
		导入测试数据：doc/dwz_springmvc.sql
		mysql -u root -p dwz_springmvc  有偿服务:

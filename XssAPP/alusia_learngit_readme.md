## 简易的命令行入门教程:
### Git 全局设置:
```
git config --global user.name "XXXX"
git config --global user.email "YYYYY"
```
### 创建 git 仓库:
```
mkdir FFFF
cd FFFF
git init
touch README.md
git add README.md
git commit -m "first commit"
git remote add origin https://gitee.com/XXXX/FFFF.git
git push -u origin master
```
### 已有项目?
```
cd existing_git_repo
git remote add origin https://gitee.com/XXXX/FFFF.git
git push -u origin master
```

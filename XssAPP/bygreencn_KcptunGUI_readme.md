### KcptunGUI是什么?
KcptunGUI是kcptun的Windows用户界面,用于帮助用户简易方便的配置参数以及管理服务.
已包含不太旧的kcptun,无需额外下载.
发布版本可能不是最新版,最新版请自行下载编译.

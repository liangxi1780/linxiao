# 简单静态企业产品展示小程序

#### 源码提供
微信小程序Club-微信小程序社区 
[www.wxappclub.com](http://www.wxappclub.com)

#### 项目介绍
企业主要产品、服务首页列表列出来，点进去是个静态页面显示详情，再来一个联系我们的页面，加入公司地址，电话，网址，可以一键拨号，一键创建联系人，一键加入美洽客服就行了

#### 安装教程

1. 下载源码
2. 用微信小程序开发工具打开
3. 设置自己的APPID
4. 上传审核

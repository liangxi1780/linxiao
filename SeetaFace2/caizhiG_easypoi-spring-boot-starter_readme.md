# easypoi-spring-boot-starter
支持spring boot 快速引入

Gradle 引入方式
``` groovy
compile group: 'cn.afterturn',name:'easypoi-spring-boot-starter',version:'4.0.0'
```

Maven 引入方式
```xml
 
     cn.afterturn 
     easypoi-spring-boot-starter 
     4.0.0 
 

```

需要spring-boot-starter-web 来支持
不需要在配置@ComponentScan("cn.afterturn.easypoi")

和easypoi版本保持一致

#OTP动态口令

#### Description
一次性密码（One Time Password，简称OTP），又称“一次性口令”，是指只能使用一次的密码。一次性密码是根据专门算法、每隔60秒生成一个不可预测的随机数字组合，iKEY一次性密码已在金融、电信、网游等领域被广泛应用，有效地保护了用户的安全。
实现otp算法的库文件，提供java、js、php版本。微信小程序搜“OTP令牌”，可以做totp验证。

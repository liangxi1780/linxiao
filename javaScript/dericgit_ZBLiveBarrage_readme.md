## 安装

#### CocoaPods
1. 在 Podfile 中添加 pod 'ZBLiveBarrage'。
2. 执行 pod install 或 pod update。
3. 具体应用请下载项目查看。

#### 手动安装
下载 ZBLiveBarrage 文件夹内的所有内容。
将 ZBLiveBarrage 下四个源文件添加(拖放)到你的工程。

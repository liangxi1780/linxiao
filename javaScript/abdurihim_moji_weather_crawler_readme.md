# 墨迹天气爬虫

#### 项目介绍
一个墨迹天气的爬虫，可爬取实时天气、实时空气质量及未来两天的天气预报


#### 安装教程

1. 安装Python3
2. 安装BeautifulSoup4、Requests、lxml

#### 使用说明

1. 访问墨迹天气官方网站https://tianqi.moji.com/
2. 找到你所在的城市
3. 分别复制天气预报和空气质量的网址到源代码的42、43行，替换原来的网址
4. 保存代码，随便扔到哪个地方，运行即可

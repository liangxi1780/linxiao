# ssm_bus
# JSP基于SSM城市公交查询系统

用户信息: 用户名,密码,姓名,性别,出生日期,身份证,籍贯,照片,家庭地址

站点信息: 记录编号,站点名称,经度,纬度

公交线路: 记录编号,线路名称,起点站,终到站,首班车时间,末班车时间,所属公司,途径站点,地图线路坐标

站站查询: 记录编号,起始站,终到站

留言信息: 记录编号,留言标题,留言内容,留言人,留言时间

新闻公告: 记录编号,标题,新闻内容,发布日期

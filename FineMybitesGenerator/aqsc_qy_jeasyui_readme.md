#jeasyui

jeasyui项目是对Easyui控件库的JSP封装，将easyui组件封装成JSP标签库。简化easyui的使用难度，学习成本，更少的代码量。友好的IDE提示，类似JSF，Asp控件的写法，提供了相对统一的事件处理机制。直接通过jsp中的java对象绑定数据到控件。

演示地址 http://jeasyui.coding.io/

# GaoQi Admin 项目

### 环境：
   - PHP 5.4+
   - MySQL 5.1+
   - Yaf 2.3.0+
   - Redis 2.6+

### 错误码
   - 格式：ABCDE
   - 说明：
      - `AB`：表示分类码，从 `10` 开始
      - `CDE`：表示当前分类下对应的错误码， 从 `100` 开始
      - `CDE`码值与 HTTP Status Code 一致时，必须表达相同含义或内容
      - `CDE`码值与Yaf的错误代码一致时，必须表达相同含义或内容

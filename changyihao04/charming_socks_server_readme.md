一个socks服务器的实现
=====================

支持socks4/5 udp代理协议的socks代理服务器实现.


##### 使用方法:

在服务器上编译运行命令

```
socks_server.exe 4567
```

在客户浏览器上设置socks代理, 主机为服务器IP, 端口为4567.

然后就可以开始使用代理服务器了.

# 史莱姆搜索
史莱姆搜索（[http://slimego.cn](http://slimego.cn)）是一款专门服务于各行业学习爱好群体查阅资料的信息分类app，目的是方便用户搜寻各类学习教程和行业资料。
正是基于这个目的，现本人也将这款完整的app开源，希望对从事移动端跨平台开发的小伙伴有所帮助。

该app采用react native + redux技术开发，可一份代码编译出android和iOS两份app，提高开发效率。

## 软件工程课程设计信息管理系统 ##

### 目的和背景 ###
>软件工程课程设计主要训练学生采用工程的概念、原理、技术和方法开发和维护软件的能力。通过独立设计和实现一个小型信息管理系统，使学生深入理解软件生存周期内若干阶段的主要任务、以及所用的方法和步骤，掌握软件生产的重要步骤、工程化的原理、方法和相关技术，提高将工程应用软件计划、开发和维护的能力。

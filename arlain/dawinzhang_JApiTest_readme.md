**运行**  
1、请先安装jdk7或jre7以上版本。 
2、解压zip文件 
3、双击运行startup.bat 
(注意:如果出现商品冲突，请使用文本编辑器修改startup.bat之中的端口值，默认8000) 
4、使用浏览器访问：http://localhost:8000

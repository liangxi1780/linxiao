## 安装

环境要求

~~~
PHP5.6以上
mysql数据库
~~~

然后复制所有文件到网站根目录


~~~
找到根目录下的daohang.sql文件导入到数据库当中
~~~

配置数据库信息
~~~
进入config/database.php 配置自己的数据库信息
~~~

设置访问目录为public即可
~~~
即刻开启自己的网址导航之旅
~~~

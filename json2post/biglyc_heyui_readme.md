## Development

You need install [hey-cli](https://www.npmjs.org/package/hey-cli).

```sh
#build
npm install
# build complete heyui
hey b
# build esm version
hey b -f hey.esm.js
# build lang js
hey b -f hey.lang.js

# develop
cd doc
npm install
hey dev

```

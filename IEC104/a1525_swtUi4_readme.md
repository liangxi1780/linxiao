# swtUI4
基于swt，jface等组件封装为一体化的java桌面应用程序框架 
从而生成跨平台的可运行，稳定的程序。 
 

 服务器自动化运维工具  

 发布升级日志  
2015年10月29日 21:30:39

2.整理maven工程，添加自动打包可执行jar插件，target目录自动生成压缩包，解压之后即可用 
3.工程任务添加，基于特定意义 
 

2015年10月14日 17:45:07升级V3.0

3.优化和晚上数据库自动备份功能，支持动态添加数据库实例,已在实际服务器运行，性能文档达到预期;

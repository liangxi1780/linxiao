# xhcms2

## 介绍
本系统由xhadmin 后台配置一键生成 包括后台管理系统 前端api接口+jwt鉴权配置 接口缓存配置
xhamdin  官网：http://www.xhadmin.com

### 框架说明
基于thinkphp6.0开发 前端框架使用H+

#### 安装说明
PHP版本最低7.1
本地安装 解析一个本地域名 设置public为运行目录
数据库配置 在 .env文件里面
jwt配置 以及 自定义配置 在config/my.php

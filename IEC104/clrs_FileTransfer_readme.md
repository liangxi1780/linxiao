### FileTansfer介绍
FileTansfer是柒软科技倾力打造的一款简单易用的文件传输系统。
### FileTansfer特性
- 多线程任务模式，易扩展
- 系统服务运行方式，重启无忧
- 支持FTP、Email、MSMQ、ActiveMQ等多种常用传输方式，可扩展
- 稳定、易用、开源

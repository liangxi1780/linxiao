我的个人blog，欢迎fork

**Feature**

- 轻量级
- 基于Markdown  
- 支持代码高亮

		```python
		print('Hello peanut')
		```

- 支持标签 

**依赖**

- markdown  将markdown文本转换成HTML
- mako    	模板引擎
- pygments	代码着色工具
 
**步骤**

1. 在`draft`目录下新建文件：`example.md`或`example.markdown`
2. 在文件开头插入：

	title:	标题(必须)  
	date:	2013-5-7  
	tag:	标签   
	tag:	标签2   

3. `python peanut.py`生成相关文件
4. `python -m SimpleHTTPServer`，访问"0.0.0.0:8000"即可预览

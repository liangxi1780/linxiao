#### 一、目录说明
1. **可以选择打开QWidgetDemo.pro一次性编译所有的，也可以进入到目录下打开pro进行编译**
2. **如果发现有些子项目没有加载请打开QWidgetDemo.pro仔细看里面的注释**
3. **编译好的可执行文件在源码下的bin目录**

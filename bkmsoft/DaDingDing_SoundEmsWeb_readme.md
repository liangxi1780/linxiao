# SoundEmsWeb
发稿端前端代码
# git 使用
* master    
    * 正式环境  ————  主分支（此分支不可push 需在测试环境中确认无误 由管理员合并分支）
* develop   
    * 测试环境  ————  测试分支（此分支为测试环境使用分支 任何开发者在自主开发结束后 都可push）
* 自定义分支 
    * 自主开发  ————  测试分支（此分支为开发分支 项目上线之后 即可删除）

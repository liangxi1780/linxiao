# Douyin_spider
抖音爬虫

# 项目博客
[Python 抖音视频和评论爬虫][5]

# 简介
定时爬取抖音首页随机视频，保存相关视频信息和评论信息到数据库，并将视频下载到本地

# 功能

 1. 爬取抖音首页视频的作者、视频ID、描述、点赞数、评论数、分享数、下载地址、背景音乐作者和名称
 2. 爬取视频对应的评论的作者、内容、点赞数、评论时间、被回复的用户名、内容、点赞数和评论时间
 3. 保存视频到本地

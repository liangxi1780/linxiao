## Spring cloud 7day simple

**7simple**总共有7个例子，展示了spring cloud的基本用法。

-------------------
7个例子包括：
* 1）一个基本的spring boot应用。
* 2）分布式配置管理服务端
* 3）分布式配置管理客户端（微服务应用）
* 4）服务注册服务端
* 5）服务注册发现客户端（微服务应用）
* 6）实现spring boot风格的web ui
* 7）使用docker发布应用

-------------------
### 开发环境
* IDE：myeclipse 10
* JDK：jdk1.7
* WINDOWS：mvn 3

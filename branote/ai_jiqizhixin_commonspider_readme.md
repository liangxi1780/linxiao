# commonspider

#### 介绍
通用蜘蛛侠 目前支持xpath css

#### 软件架构
软件架构说明 scrapy(这个单词真难拼写)


#### 安装教程

1.  pip install scrapy
2.  scrapy startproject mydemo
3.  scrapy genspider mydemo-spider
4.  scrapy crawl mydemo-spider
5.  pip install pymongo
#### 使用说明

1.  在xx_spider.py中设置爬虫网址
2.  xxxx
3. 运行start.py

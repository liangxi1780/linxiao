# 人脸标注脚本

#### 介绍
用于训练智能教学项目的 人脸数据标注 C++ Opencv 开发
对一段视频进行人脸标注 设置锚点。
使用时需要先在内部设置你的视频路径。
结果可以在截取图像文件集合文件中找到

#### 软件架构
C++ opencv VS2015以上版本


#### 安装教程
使用前需要配置opencv3.x

#### 使用说明
空格暂停 
鼠标选取ROI区域
按S继续

# Event-recommend
数据说明：Event Recommendation Engine Challenge 是 Kaggle 上的一个推荐任
务竞赛，根据用户的历史参加活动记录、用户的社交信息、以及用户历史上在 App
上浏览和点击的活动，预测用户是否会对某个活动感兴趣

# Per.js

#### 项目介绍
Per.js - 快速、简便的响应式JavaScript开发框架。

- 他可以有效的帮助你减少需要编写的代码量
- 他完全是开源可扩展的
- 他的执行速度几乎是Vue.js的8~7分之一倍

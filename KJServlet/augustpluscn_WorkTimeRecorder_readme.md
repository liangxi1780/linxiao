#WorkTimeRecorder
 简述 
WTR是方便那些每天都需要记录工作内容和时间的ITers而开发的小工具。 
涵盖了员工工号、电脑编号（名称）、员工分级、资产编号、问题简述和处理时间这些栏位。 
对于长期在HelpDesk领域工作而又需要记录日常工作内容的朋友非常有用。

# draggable-example

> Example usage of [vue.draggable](https://github.com/SortableJS/Vue.Draggable)

![demo gif](example.gif)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

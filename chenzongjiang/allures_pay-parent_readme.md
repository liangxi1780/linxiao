# pay-parent

#### 介绍
金融聚合支付平台系统。主要有管理平台、代理商平台、商户平台；聚合各大支付通道，提供风控、线上自动化配置、结算、支付、权限框架；核心技术：spirngboot 、springcloud、redis、rabbitMQ、quartz；目前项目正在进一步完善中，版本上线2019年10月1日上线；仅供学习，对支付感兴趣的童鞋可以留言，拉群；

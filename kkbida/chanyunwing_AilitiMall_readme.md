## python虚拟环境各模块版本
避免共同开发各版本工具造成的错误，请务必同意开发工具的版本，别擅自使用不同版本开发。
请使用`requirements.txt`中的版本
创建虚拟环境后，将`requirements.txt`放在当前环境目录下
敲命令：`pip install -r requirements.txt`
